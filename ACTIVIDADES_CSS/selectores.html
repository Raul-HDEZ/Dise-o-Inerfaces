Pseudo-clases

Con CSS se puede añadir sombra al texto o elemento seleccionado. Hay dos propiedades importantes dentro del sombreado de elementos: text-shadow y box-shadow.
La primera aplica texto a un texto, y se le pueden especificar tanto los pixeles que se moverá el sombreado desde la posición del texto orignial en ambos ejes (x e y), como los que ocupará el difuminado y el color del mismo, entre otras cosas.
La segunda, por otro lado, sirve para dar sombras a otros elementos del html. Se declara de la misma manera que la anterior propiedad.

Pseudo-elementos

Existen cuatro propiedades dentro de este epígrafe:

Text-overflow
Esta propiedad especifica la manera en la que se le señala al usuario el contenido desbordado que no se muestra.

Word-wrap
Permite forzar el ajuste de texto incluso si se ha de partir una palabra. Esto puede darse cuando el tamaño de una palabra es superior al de la caja que la contiene.

Word-break
Esta propiedad especifica las reglas de salto de línea de un texto determinado.

Writing-mode
Especifica si las líneas de texto escogidas se distribuyen a lo largo de un área determinada de manera vertical u horizontal.
