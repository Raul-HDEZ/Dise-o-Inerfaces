Rounded Corners

Con la propiedad border radius se le pueden otorgar esquinas redondeadas a un elemento. Esta propiedad define el radio de las esquinas del elemento en cuestión. Además, se puede elegir qué esquinas redondear y cuánto, para darle un toque diferente, y no tienen por qué ser redondas: también pueden ser elípticas.

Border Images

Con la propiedad border-image, se puede se puede elegir una imagen para ser utilizada como el borde que rodea a un elemento determinado. Tiene tres partes: qué imagen usar como borde, dónde cortarla y si las secciones intermedias deben repetirse o estirarse para conformar el borde.

Backgrounds

CSS permite añadir múltiples imágenes de fondo mediante la propiedad background-image. Los diferentes fondos se separan por comas y las imágenes seleccionadas se van superponiendo unas encima de otras, de manera que la primera imagen queda más "cerca" del usuario.
Por otro lado, existe la propiedad background-size, que permite especificar el tamaño de las imágenes de fondo. El tamaño se puede espcificar en longitudes, porcentajes, o utilizando una de las dos palabras clave: contain o cover.
La palabra clave contain sirve para que el tamaño de la imagen sea lo más grande posible dentro del área definida, haciendo que tanto la altura como la anchura entren en ella.
Por otro lado, la palabra clave cover sirve para que la imagen cubra completamente el área seleccionada, aunque algunas partes se queden fuera.

Gradients

Los degradados o gradients permiten realizar transiciones suaves entre dos o más colores.
Se puede degradar el color de manera linear, utilizando linear-gradient, especificando tanto el tipo (horizontal, vertical, diagonal) como la dirección (de arriba hacia abajo, de izquierda a derecha...); utilizando ángulos, utilizando la etiqueta anterior con el añadido angle antes de la declaración de los colores, y usando paradas de varios colores.

Shadows

Con CSS se puede añadir sombra al texto o elemento seleccionado. Hay dos propiedades importantes dentro del sombreado de elementos: text-shadow y box-shadow.
La primera aplica texto a un texto, y se le pueden especificar tanto los pixeles que se moverá el sombreado desde la posición del texto orignial en ambos ejes (x e y), como los que ocupará el difuminado y el color del mismo, entre otras cosas.
La segunda, por otro lado, sirve para dar sombras a otros elementos del html. Se declara de la misma manera que la anterior propiedad.

Text Effects

Existen cuatro propiedades dentro de este epígrafe:

Text-overflow
Esta propiedad especifica la manera en la que se le señala al usuario el contenido desbordado que no se muestra.

Word-wrap
Permite forzar el ajuste de texto incluso si se ha de partir una palabra. Esto puede darse cuando el tamaño de una palabra es superior al de la caja que la contiene.

Word-break
Esta propiedad especifica las reglas de salto de línea de un texto determinado.

Writing-mode
Especifica si las líneas de texto escogidas se distribuyen a lo largo de un área determinada de manera vertical u horizontal.
