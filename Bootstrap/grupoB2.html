Alerts



Las alertas funcionan como una clase para los div utilizando el atributo class y el atributo role ademas de una subclase. Por ejemplo, divs con la class alert y el role alert con las distintas subclases de alerts en bootstrap 5.

Secondary alert!

Danger alert!
Warning alert!
Info alert!

Botones y grupo de botones



Bootstrap 5 proporciona diferentes estilos de botones para acciones en formularios, cuadros de diálogo y más con soporte para múltiples tamaños, estados y más.

Estilo de botones

Las clases de botones se pueden usar en elementos a , button o input

Link IES Tetuan

<div>
<button type="button" class="btn btn-primary">Primary</button>
<button type="button" class="btn btn-danger">Danger</button>
<a href="https://www.iestetuan.es/cmsj/" class="btn btn-success" target="blank">Link IES Tetuan</a>
</div>

Contorno del botón

Reemplace las clases de modificador predeterminadas con las .btn-outline-* que eliminan todas las imágenes y colores de fondo en cualquier botón.


<div>
<button type="button" class="btn btn-outline-primary">Primary</button>
<button type="button" class="btn btn-outline-danger">Danger</button>
</div>

Tamaño de los botones

Use la .btn-lg clase para botones grandes o .btn-sm la clase para botones pequeños:


<div>
<button type="button" class="btn btn-primary btn-lg">Large</button>
<button type="button" class="btn btn-primary">Default</button>
<button type="button" class="btn btn-primary btn-sm">Small</button>
</div>

Botones de nivel de bloque

Para crear un botón de nivel de bloque que abarque todo el ancho del elemento principal, use la .d-grid clase "ayudante" en el elemento principal. Si tiene muchos botones a nivel de bloque, puede controlar el espacio entre ellos con la .gap-* clase:


<div>
<div class="d-grid gap-3">
<button type="button" class="btn btn-primary btn-block">Full-Width Button</button>
<button type="button" class="btn btn-primary btn-block">Full-Width Button</button>
<button type="button" class="btn btn-primary btn-block">Full-Width Button</button>
</div>
</div>

Botones activos/deshabilitados

Un botón se puede establecer en un estado activo (aparece presionado) o deshabilitado (no se puede hacer clic). La clase .active hace que un botón parezca presionado y el disabled atributo hace que no se pueda hacer clic en un botón.


<div>
<button type="button" class="btn btn-primary active">Active Primary</button>
<button type="button" class="btn btn-primary" disabled>Disabled Primary</button>
</div>

Badges



Una badge o insignia o insignias son etiquetas (label) que en bootstrap funcionan con una clase badge" y una subclase, en este caso "badge-secondary"

UN TITULO una badge

Las badges escalan hasta el tamaño del elemento padre. En el ejemplo tenemos 3 titulos de distintos tamaños con el mismo badge que escalan a el mismo tamaño que el elemento titulo del que descienden.

SOY UN TITULO PEQUEÑO badge pequeño

SOY UN TITULO badge

SOY UN TITULO GRANDE badge grande

<h1> SOY UN TITULO GRANDE <span class="badge bg-secondary"> badge grande </span> </h1>

Las badges pueden ir dentro de elementos como por ejemplo button.

Las badges pueden tambien tener tambien otra posición relativa al elemento que las contiene.



<button type="button" class="btn btn-primary position-relative" >
Inbox <span class="position-absolute top-0 start-100 translate-middle badge rounded-pill bg-danger"> 99+ </span>
</button>

Bootstrap 5 cuenta con 8 subclases de badged en función del color.

Primary Secondary Success Danger Warning Info Light Dark

<span class="badge bg-primary"> Primary </span>

Por ultimo las badge pueden redondearse usando la clase rounded-pill y el texto puede cambiarse a negro usando la clase text-dark.

Progress Bars



Basic Progress Bar

Las Progress Bars se utilizan para mostrar al usuario como va un progreso
para crear una progress bar usamos la clase progress al contenedor y la clase progress-bar al div hijo

<div class="progress"> <div class="progress-bar" style="width:70%"></div> </div>



Etiquetas

Se puede añadir texto dentro de las progress bars

<div class="progress"> <div class="progress-bar" style="width:70%">70%</div> </div>

70%


Colores

Se utiliza la propiedad de fondo para agregar un color a la progress bar como background-color: o bg- de bootstrap

<div class="progress"> <div class="progress-bar bg-danger" style="width:50%">Rojo</div> </div><br>

Azul

Verde

Turquesa

Amarillo

Rojo

Striped Progress Bars

Se añade la clase progress-bar-striped para crear una progress bar rallada

<div class="progress"> <div class="progress-bar progress-bar-striped" style="width:40%"></div> </div>



Animaciones

Se pueden crear progress bar animadas añadiendo la clase progress-bar-animated

<div class="progress"> <div class="progress-bar progress-bar-striped progress-bar-animated" style="width:40%"></div> </div>



Multiple

Las progress bar tambien se pueden poner una a continuacion de la otra poniendolas todas dentro del mismo div con la clase progress

<div class="progress-bar bg-danger" style="width:20%">Peligro </div>
</div>

Spinners


Border spinner

Los spinners se utilizan para mostrar al usuario el estado de carga en la web
para crear un spinner usamos la clase spinner-border

<div class="spinner-border" role="status"> <span class="visually-hidden">Cargando...</span> </div>

Cargando...


Colores

Se utiliza la propiedad de color de texto para agregar un color al spinner como color: o text- de bootstrap

<div class="spinner-border text-danger" role="status"></div>



Growing spinners

Son spinners que en vez de girar crecen, se utiliza spinner-grow y utilizan la propiedad de color del texto al igual que los anteriores

<div class="spinner-grow"></div>



Tamaño

Se puede ajustar el tamaño de los spinners usando las clases spinner-border-sm y spinner-grow-sm
tambien se puede usar css para determinar sus dimensiones con width: y height:

<div class="spinner-border spinner-border-sm" role="status"></div>
<div class="spinner-grow spinner-grow-sm" role="status"></div>



Botones

Tambien se pueden añadir spinners a los botones

<button class="btn btn-primary"> <span class="spinner-border spinner-border-sm"></span> Cargando... </button>
